# Design Concept

## 1. 設計目標
本專案以音樂品牌形象為核心，將原有 WordPress 網站重新整理為乾淨的前端架構，
並強化視覺層級與內容呈現，打造簡潔、現代且具有視覺張力的網站介面。

## 2. 設計風格
- 極簡風格（Minimalism）
- 強調留白與內容聚焦
- 大標題搭配形象視覺圖片
- 區塊式排版提升閱讀流暢度

## 3. 使用體驗（UX）
- 清楚的導覽結構（Home / About / Service 等）
- 多語系切換（中文 / 英文 / 日文）
- 響應式設計（Desktop / Tablet / Mobile）
- 強化可讀性與資訊層級

## 4. 視覺元素
- 主色：橘色（品牌主視覺）
- 輔助色：黑 / 白，提升對比與辨識度
- 字體：系統字體搭配 Serif 標題，營造品牌感

## 5. 設計與實作流程
本專案初期透過 WordPress Elementor 編輯器建立頁面結構，
再轉換為純前端程式碼進行優化與重構，
最後補充 Figma UI 設計稿作為視覺與結構整理依據。

## 6. UI 設計工具
UI 設計稿使用 Figma 製作，依據實際前端畫面進行整理與還原。
